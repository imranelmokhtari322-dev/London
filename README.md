# IPTV London

Premium UK IPTV streaming service website. Built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Dark Netflix-style UI with film grain and cinema aesthetics
- Live UK channel preview with EPG progress bars
- Trending films carousel (UK Box Office)
- Pricing plans with checkout modal
- Broadband speed/compatibility tester
- Device setup guides (Firestick, Smart TV, Apple, Android)
- FAQ accordion
- Channel logos marquee slider
- Fully responsive

## Tech Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Framer Motion (motion/react)
- Lucide React icons

## Run Locally

**Prerequisites:** Node.js 18+

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```
